¿Qué es una sucesión geométrica?

Una sucesión geométrica es una secuencia de números donde cada término después del primero se obtiene multiplicando el término anterior por una constante llamada razón (r).

Ejemplo

Para la sucesión: 2, 6, 18, 54, 162,...

El primer término (a₁) es 2 y la razón (r) es 3 (porque 6/2 = 3, 18/6 = 3, etc.).

Fórmula del término general

El término general de una sucesión geométrica se calcula con:

aₙ = a₁ × r(n-1)

Donde:

  • aₙ = término n-ésimo
  • a₁ = primer término
  • r = razón
  • n = número del término a calcular
Cálculo

Para encontrar el 5° término de la sucesión 2, 6, 18,...

a₅ = 2 × 3(5-1) = 2 × 34 = 2 × 81 = 162

Suma de los primeros n términos

La suma de los primeros n términos de una sucesión geométrica se calcula con:

Sₙ = a₁ × (1 - rn) / (1 - r) (si r ≠ 1)

Cálculo

Para sumar los primeros 4 términos de 2, 6, 18, 54,...

S₄ = 2 × (1 - 34) / (1 - 3) = 2 × (1 - 81) / (-2) = 2 × (-80) / (-2) = 80

Comprobación: 2 + 6 + 18 + 54 = 80

Suma de una serie geométrica infinita

Si |r| < 1, la suma infinita converge a:

S = a₁ / (1 - r)

Ejemplo

Para la sucesión 1, 1/2, 1/4, 1/8,... donde a₁=1 y r=1/2:

S = 1 / (1 - 1/2) = 1 / (1/2) = 2
